Simple Fuzzing Lib
---

C Library containing some simple functions to generate random inputs used for fuzzing.

- do_maybe(function * f, probability p)
- random_in_between(start, end)
- random_b()
- random_w()
- random_dw()
- random_qw()
- mutate_b(byte& b)
- mutate_w(word& w)
- mutate_dw(doubleword& dw)
- mutate_qw(quadword& qw)

- rand_init()
